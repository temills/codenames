Descriptions of the files:

--------------

dynamicComputeWordSemanticsBasedonContexts.py: the python code to show the dynamics of word semantics based on different contexts.

DICTIONARY.txt: the dictionary

run.sh: the script code to run.

wordsWithContexts.test: the test words with different contexts. Seperate the word and its context with a “#”
-------------

please run the run.sh in your shell by typing the following command.

#sh run.sh

Python 3.5 is needed.
g++ is needed.
